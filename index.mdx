# CapyUniverse Docs

**O hub onde projetos deixam de ser arquivos soltos e passam a ter contexto, padrão, rastreabilidade e evolução.**

CapyUniverse é a **plataforma-mãe local** do ecossistema Capy. Ela organiza ferramentas com IA, agentes especializados, produtos digitais, protótipos, conhecimento técnico e documentação em um único universo operacional.

## Definição do produto

**CapyUniverse é um hub local para criar, organizar, testar, documentar e evoluir projetos digitais com IA.**

Ele não é apenas um website com várias páginas. A visão correta é tratá-lo como um **universo modular**: cada ferramenta, agente ou projeto filho precisa ter objetivo, contexto, stack, status, limitações, fonte canônica e próximos passos.

<Info>
  Esta documentação prioriza evidência. Quando uma informação não aparece no repositório, no CSV ou em fonte fornecida, ela deve ficar marcada como **INFORMAÇÃO NÃO FORNECIDA**.
</Info>

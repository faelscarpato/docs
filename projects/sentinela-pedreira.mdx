## Resumo

Sentinela Pedreira é uma plataforma pública de inteligência cívica municipal que organiza, analisa e disponibiliza documentos públicos de forma acessível e estruturada.

## Papel dentro do CapyUniverse

É um projeto de alto impacto público. Dentro do ecossistema, representa a frente de **IA aplicada à transparência, documentos públicos e análise cívica**.

## Stack observada

- React.
- TypeScript.
- Vite.
- react-router.
- Tailwind/CSS utilitário.
- react-markdown.
- remark-gfm.
- Supabase presente na estrutura do projeto.

## Fluxo documental identificado

```txt
Documentos públicos
  ↓
Normalização / base de dados
  ↓
Cards e páginas de consulta
  ↓
Análises Markdown
  ↓
Renderização acessível na interface
```

## Riscos principais

| Risco | Mitigação |
| --- | --- |
| Interpretação incorreta de documento público | Separar fato, análise e opinião |
| Atualização incompleta | Criar rotina de ingestão e revisão |
| Responsabilidade pública | Exibir fontes, datas e limites da análise |
| Dados sensíveis | Curadoria antes de publicação |
| Viés de IA | Revisão humana e transparência metodológica |

## Evolução recomendada

1. Criar página de metodologia.
2. Padronizar formato das análises.
3. Adicionar fonte, data e status em cada documento.
4. Criar alertas de atualização.
5. Criar busca avançada.
6. Criar seção “Como interpretar”.
7. Integrar CapyMind como memória documental.

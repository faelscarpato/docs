## Resumo

O CapyUniverse Core é o centro do ecossistema. Ele organiza ferramentas com IA, páginas funcionais, agentes especializados, experimentos e projetos filhos.

## Objetivo

Criar uma base local e modular para:

- testar ferramentas com IA;
- organizar prompts e agentes;
- acelerar criação de produtos;
- documentar tudo em um universo único;
- servir como portfólio técnico vivo.

## Stack observada

- HTML.
- CSS.
- JavaScript.
- React.
- Vite.
- Integrações planejadas/indicadas com Gemini e OpenAI.

## Fluxo conceitual

```txt
Usuário → Interface Web → CapyTool/Agente → Serviço IA → Resposta → Histórico/estado local
```

## Módulos esperados

| Módulo | Função |
| --- | --- |
| Navegação central | Organizar acesso às ferramentas |
| CapyTools | Páginas/ferramentas independentes |
| Serviços IA | Camada comum para provedores |
| Catálogo de projetos | Índice dos projetos filhos |
| Documentação | Fonte oficial do ecossistema |

## Roadmap específico

1. Criar catálogo unificado de CapyTools.
2. Padronizar estrutura de módulo.
3. Separar prompts mestres por agente.
4. Criar sistema de configuração de provedores IA.
5. Adicionar persistência local padronizada.
6. Evoluir para modo plugin ou monorepo.
7. Conectar CapyMind como base de contexto.

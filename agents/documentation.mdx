---
title: "Agente de Documentação"
description: "Agente para criar, revisar e manter documentação dos projetos Capy."
---

O Agente de Documentação é responsável por transformar projetos, prompts, decisões e código em documentação clara e sustentável.

## Objetivo

Manter o CapyUniverse organizado como ecossistema documentado.

## Tarefas principais

- Criar páginas Mintlify.
- Padronizar páginas de projetos filhos.
- Revisar links e navegação.
- Transformar README em documentação estruturada.
- Criar glossário e padrões.
- Gerar checklist de qualidade.
- Manter roadmap e changelog.

## Critérios editoriais

| Critério | Regra |
| --- | --- |
| Clareza | Explicar para quem é e para que serve |
| Veracidade | Não inventar funcionalidades |
| Prioridade | Destacar o que é uso real e o que é futuro |
| Manutenção | Cada página deve ter dono ou contexto |
| Escalabilidade | Estrutura deve comportar novos projetos |

## Fluxo de revisão

```txt
Nova página
  ↓
Revisão de objetivo
  ↓
Revisão técnica
  ↓
Revisão de links
  ↓
Checklist de qualidade
  ↓
Publicação
```

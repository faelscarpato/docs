---
title: "Template de projeto filho"
description: "Modelo para documentar novos projetos conectados ao CapyUniverse."
---

Use este template sempre que um projeto filho entrar no ecossistema.

```md
---
title: "[Nome do projeto]"
description: "[Descrição curta]"
---

## Resumo

## Papel dentro do CapyUniverse

## Problema que resolve

## Público-alvo

## Funcionalidades atuais

## Stack

## Arquitetura resumida

## Como rodar

## Limitações atuais

## Riscos

## Roadmap

## Fonte do projeto
```

## Campos obrigatórios

| Campo | Por quê |
| --- | --- |
| Resumo | Ajuda qualquer pessoa a entender rápido |
| Papel no universo | Evita projeto solto |
| Funcionalidades atuais | Separa fato de ideia |
| Limitações | Evita promessa indevida |
| Roadmap | Direciona evolução |
| Fonte | Mantém rastreabilidade |

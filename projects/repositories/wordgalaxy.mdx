# Word Galaxy

## Resumo verificado

Word Galaxy é descrito no README como um jogo de palavras de nova geração, inspirado em aventuras de cruzadinhas, com exploração espacial, desafios linguísticos, mundos temáticos e geração procedural de puzzles.

## O que existe no README

- Mecânica de cruzadinhas interconectadas.
- Exploração por mundos temáticos.
- Feedback sensorial com animações, sons e partículas.
- Geração procedural de puzzles.
- Uso de Web Workers para geração em tempo real.
- Estrutura de telas como Splash, Home, WorldMap, Shop e Ranking.

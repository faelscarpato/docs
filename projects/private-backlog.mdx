O CSV enviado contém **32 repositórios privados**.

Como a documentação do CapyUniverse está online, esta página **não publica nomes, URLs nem descrições dos projetos privados** para evitar exposição acidental.

## Como documentar privados com segurança

1. Criar uma área privada ou repositório separado de documentação interna.
2. Publicar apenas o nome comercial se ele já for público.
3. Remover URLs de repositórios privados de qualquer página pública.
4. Separar descrição pública de detalhes técnicos internos.
5. Validar com o mantenedor antes de liberar qualquer página privada.
